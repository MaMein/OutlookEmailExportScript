# OutlookEmailExportScript
Exports Emails to harddrive using the folderstructure from EMail Database.

## For configuration use the following modul constants
+ Export from main accounts email folder with the name = CONST STRING_NAME_EXPORT_EMAILFOLDER
+ Target Folder on Harddrive for export files = CONST STRING_FOLDERNAME_TARGET
+ Name of the Table-of-Contents file = Const STRING_FILENAME_TOC
